Select waypoints here. Enter at least 3 characters to search for a airport or waypoint as listed in the En Route Supplement as published by CASA. Then select a valid altitude from the list (using the CASA format)
The lookup is via web api backend currently hosted in Azure.

The waypoints will be drawn on the (google) map, and the resulting flight path calculations appear on the "Plan Details" tab. The map will be updated as waypoints are added or removed.
